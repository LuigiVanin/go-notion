# Go Writable 📒

An simple notes application similar to notion. Backend created with golang using go-fiber and gorm. Using vue.js + vite on the frontend.

## Run With Docker 🐋

```bash
> docker build -t go-writable .
> docker run -p 3000:3000 go-writable
```
